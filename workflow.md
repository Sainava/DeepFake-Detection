## üéØ **Critical Decision Points**

1. **Frame Sampling**: Fixed 32 frames ‚Üí Uniform temporal coverage across video lengths
2. **Flow Algorithm**: TV-L1 ‚Üí Accurate dense motion estimation between consecutive frames  
3. **CNN Architecture**: Dual ResNet18 ‚Üí RGB spatial + Flow motion feature fusion
4. **Feature Fusion**: Concatenation ‚Üí 1024D joint representation (512+512)
5. **Temporal Modeling**: Bi-LSTM ‚Üí Forward + backward sequence classification
6. **Training Strategy**: CNN first, then LSTM ‚Üí Staged learning for stability

## üìä **DETAILED COMPONENT BREAKDOWN**

### **MixedResNet Architecture Details**
```python
# RGB Branch: ResNet18 ‚Üí 512D features
# Flow Branch: ResNet18 ‚Üí 512D features  
# Fusion: Concatenate(RGB, Flow) ‚Üí 1024D
# Head: Dropout(0.5) ‚Üí Linear(1024, 2) ‚Üí Softmax
```

### **Bi-LSTM Configuration** 
```python
# Input: (Batch, Sequence_Length, 1024)
# LSTM: 2 layers, 512 hidden, bidirectional=True
# Output: Hidden states ‚Üí Dropout(0.5) ‚Üí Linear(1024, 2)
```

### **Training Hyperparameters**
| Parameter | MixedResNet | Bi-LSTM |
|-----------|-------------|---------|
| **Batch Size** | 8 | 16 |
| **Learning Rate** | 1e-4 | 1e-3 |
| **Epochs** | 12 | 30 |
| **Optimizer** | Adam | Adam |
